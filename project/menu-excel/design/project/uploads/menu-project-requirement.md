# 카페 메뉴판 (Excel 구동)

Excel 파일 하나로 메뉴판 화면을 구동한다. **Excel의 탭(시트) 1개 = 메뉴판 페이지 1개**로 매핑되고,
**Excel을 수정하고 저장하면 화면이 즉시 갱신**된다(파일 감시 + SSE).

이 프로젝트는 강의 앵커 예제다: "데이터(Excel)와 표현(템플릿)을 분리한다"는 원칙을 한 화면에서 보여준다.

---

## 빠른 시작

```bash
# 1) 의존성 설치
npm install

# 2) 서버 실행
npm start
# → http://localhost:3000 접속

# 3) menu.xlsx를 Excel에서 열어 값을 바꾸고 "저장(Ctrl+S)"
#    → 브라우저 메뉴판이 자동으로 갱신된다
```

> 주의: 값을 바꿔도 **저장(Ctrl+S) 전까지는 반영되지 않는다.** 로컬 .xlsx는 저장 시점에만 디스크에 기록되기 때문이다. (OneDrive 자동저장이면 즉시 반영)

## Excel 작성 규칙

### 메뉴 시트 (커피 / 디저트 / 음료 …)

첫 행은 헤더, 시트 이름이 곧 페이지(탭) 이름이 된다.

| 메뉴명 | 가격 | 설명 | 품절 | 카테고리 |
|---|---|---|---|---|
| 아메리카노 | 4500 | 깔끔한 산미 | | 에스프레소 |
| 디카페인 아메리카노 | 5000 | 스위스 워터 디카페인 | Y | 에스프레소 |

- **가격은 숫자만** 입력한다(₩·콤마 금지). 통화 서식은 화면에서 `원`을 붙여 표시한다.
- **품절은 `Y`** 또는 빈칸. `Y`면 취소선 + 흐림 + "품절" 라벨로 표시된다.
- 시트(탭)를 추가하면 메뉴판 페이지가 자동으로 늘어난다.

### `_설정` 시트 (언더스코어로 시작 → 페이지에서 제외)

| 항목 | 값 |
|---|---|
| 매장명 | 빌런 커피 |
| 테마 | cafe-dark |
| 자동전환초 | 0 |

- **테마**: `cafe-dark` 또는 `bistro-light` (코드 수정 없이 Excel에서 디자인 변경)
- **자동전환초**: 0이면 수동, 양수면 그 초마다 페이지 자동 전환(디지털 사이니지용)

## 동작 원리

```text
점주가 Excel 수정 → 저장
        │
        ▼
menu.xlsx ──(chokidar 파일 감시, 300ms 디바운스)──▶ server.mjs
        │                                              │
        │                              SSE: event: menu-updated
        ▼                                              ▼
   재파싱(JSON)  ◀── GET /api/menu ──  브라우저(메뉴판) ── 선택 테마로 재렌더
```

- **server.mjs**: `_` 접두사가 아닌 시트를 페이지로 변환, `_설정`에서 매장명·테마·자동전환초를 읽는다.
- **SSE(Server-Sent Events)**: 메뉴판은 읽기 전용 단방향 표시이므로 WebSocket 대신 SSE로 충분하다.
- **디바운스 300ms**: Excel 저장 시 발생하는 중복 파일 이벤트를 한 번으로 합친다.

## SheetJS(xlsx) 설치 주의

`package.json`은 SheetJS를 **공식 CDN**에서 받도록 지정한다.

```json
"xlsx": "https://cdn.sheetjs.com/xlsx-0.20.3/xlsx-0.20.3.tgz"
```

> npm 레지스트리의 `xlsx`는 0.18.5로 4년 묵은 구버전이다. SheetJS는 배포를 자체 CDN으로 이전했으므로 `npm install xlsx`(레지스트리)가 아닌 위 CDN URL을 사용한다. 보안 스캐너(Snyk 등)가 SheetJS에 오탐을 내는 알려진 이슈가 있으나 공식적으로 무시 권장이다.

## 응용 아이디어 (강의 확장)

- 템플릿 CSS를 추가해 테마를 늘린다(같은 데이터, CSS만 교체).
- `카테고리` 열로 한 페이지 안에서 소제목 그룹을 나눈다.
- HTTP 공유 방식: 사내 파일서버에 menu.xlsx를 올리고 브라우저가 폴링(ETag)으로 갱신.
- 키오스크: 브라우저를 전체화면으로 띄우고 `자동전환초`로 페이지를 순환시킨다.
