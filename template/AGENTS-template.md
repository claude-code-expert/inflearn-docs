## 도구별 호환성 (선택)

### Claude Code 사용 시

루트에 `CLAUDE.md`를 만들고 다음 한 줄로 본 AGENTS.md를 참조한다 (심볼릭 링크보다 권장).

```markdown
<!-- CLAUDE.md -->
@AGENTS.md 의 모든 지침을 따른다.

## Claude Code 전용 추가
[Hook 연동·서브에이전트·Skill 등 Claude 전용 설정]
```

### Cursor 사용 시

`.cursor/rules/*.mdc` 형식 사용. `.cursorrules`(legacy)는 Agent 모드에서 무시되므로 쓰지 말 것.

### OpenAI Codex 사용 시

본 AGENTS.md를 그대로 사용. 모노레포 하위 폴더에 다른 규칙이 필요하면 `<폴더>/AGENTS.override.md`로 오버라이드.
